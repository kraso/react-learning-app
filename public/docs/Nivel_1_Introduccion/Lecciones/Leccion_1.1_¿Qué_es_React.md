## Lección 1.1: ¿Qué es React?

**Contenido de la lección:**

React es una biblioteca de JavaScript para construir interfaces de usuario. Fue desarrollada por Facebook (ahora Meta) y es una de las librerías más populares para el desarrollo de
aplicaciones web modernas.

React se basa en componentes reutilizables que permiten construir IU (interfaces de usuario) complejas de manera eficiente. Es una biblioteca de vista (frontend) que puede ser utilizada con
otras librerías para completar una aplicación full-stack.

**Características principales de React:**
- Componentes reutilizables
- Uso de Virtual DOM para mejorar el rendimiento
- Una sola dirección de datos (unidireccional)
- Gran ecosistema y comunidad activa
- Desarrollo declarativo y componentizado

**¿Por qué aprender React?**
- Es ampliamente usado en la industria
- Tiene una gran comunidad y documentación
- Fácil integración con otras bibliotecas
- Soporte continuo por parte de Meta
- Alta demanda laboral
