## Lección 1.2: Ventajas de usar React

**Contenido de la lección:**

- **Componentes reutilizables:** Los componentes permiten construir elementos de interfaz una sola vez y reutilizarlos en diferentes partes de la aplicación.

- **Virtual DOM:** React utiliza un DOM virtual para mejorar el rendimiento al minimizar las operaciones en el DOM real.

- **Una sola dirección de datos:** Los datos fluyen en una única dirección, lo que hace que la aplicación sea más predecible y fácil de depurar.

- **Gran ecosistema y comunidad activa:** React tiene una gran comunidad que contribuye con bibliotecas adicionales, herramientas y documentación.

- **Desarrollo declarativo:** Se describe qué debería suceder en lugar de cómo lograrlo paso a paso.

- **Soporte para testing:** Facilita la creación de tests unitarios y de integración.

- **Desarrollo flexible:** Puede usarse en proyectos pequeños y grandes, desde componentes individuales hasta aplicaciones completas.
