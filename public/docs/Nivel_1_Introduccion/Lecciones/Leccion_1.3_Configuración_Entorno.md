## Lección 1.3: Configuración del entorno de desarrollo

**Contenido de la lección:**

Para comenzar con React, necesitas:

1. **Instalar Node.js** (versión 14 o superior):
    - Descarga desde https://nodejs.org/
    - Verifica la instalación con los comandos:
    ```bash
    node --version
    npm --version
    ```

2. **Crear un nuevo proyecto React**:
    ```bash
    npx create-react-app mi-app
    cd mi-app
    npm start

3. Estructura básica de un proyecto React:
    - src/ - Código fuente
    - public/ - Archivos estáticos
    - package.json - Dependencias y scripts
    - README.md - Documentación del proyecto

4. Editor de código recomendado:
    - Visual Studio Code
    - Configuración con extensiones:
    ES7 React/Redux/GraphQL/React-Native snippets
    Auto Rename Tag
    Prettier

5. Herramientas útiles:
    - React Developer Tools (extensión de navegador)
    - Chrome DevTools
    - ESLint para validación de código
