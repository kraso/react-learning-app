## Ejercicio 1: Hook personalizado

Crea un hook personalizado llamado usarLocalStorage que permita gestionar un valor en el almacenamiento local.

Requisitos:
- Debe aceptar una clave y un valor inicial
- Debe devolver el valor actual y una función para actualizarlo
- Debe cargar el valor desde localStorage si existe
- Debe guardar el valor en localStorage cuando cambia
